🗳️ Online Voting System (Frontend Simulation)

📌 Overview

This project is a simple Online Voting System developed using HTML, CSS, and JavaScript. It simulates a digital voting interface where users can select a candidate and cast their vote. The system ensures that each user can vote only once and displays the voting results.

🎯 Objective

The objective of this project is to create awareness about digital voting systems by simulating a user-friendly voting interface.


🚀 Features

- Candidate selection
- One vote per user logic
- Result display
- Form validation
- Responsive UI design


🛠️ Technologies Used

- HTML – Structure of the application
- CSS – Styling and layout
- JavaScript – Logic and DOM manipulation

⚙️ How It Works

Users select a candidate and submit their vote. The system validates the input and ensures that only one vote is allowed per user. After voting, the results are displayed dynamically.

📂 Project Structure

- index.html – Contains HTML, CSS, and JavaScript code

- Building interactive web applications
